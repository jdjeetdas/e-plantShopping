# e-plantShopping

Paradise Nursery - Online Plant Shopping Application

A React-based shopping cart application for an online plant nursery that offers a variety of house plants. Built with React, Redux Toolkit, and Vite.

## Features

- Landing page with company information and Get Started button
- Product listing page with plants grouped into categories
- Shopping cart with add/remove/update quantity functionality
- Dynamic cart icon showing total items
- Redux state management for cart operations

## Tech Stack

- React 18
- Redux Toolkit
- Vite
- CSS3

## Getting Started
```bash
npm install
npm run dev
```
